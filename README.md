This app is designed to explore National Parks in the United States with a focus on African-American Hertitage. Users may use the app to get an overview of the parks' themes and save the parks to a dashboard to plan a trip. This project was created with a Ruby on Rails backend and a React frontend.

The application logic is contained in the controllers.
